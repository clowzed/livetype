# Livetype
A javascript library with one function which imitates typing text


# Installation
```html
<script src="https://cdn.jsdelivr.net/npm/livetype@1.0.0/index.js"></script>
```

# Usage
```html
<body onload="livetype(document.body);"> <!-- Or use any other node -->
```
